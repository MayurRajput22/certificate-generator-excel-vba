# 📜 Certificate Generator using Excel VBA

## Overview

This project automates the certificate generation process using Microsoft Excel VBA and Microsoft Word. It reads candidate details from an Excel workbook, replaces placeholders in a Word certificate template, and generates personalized PDF certificates automatically, eliminating repetitive manual work.

---

## Features

- Select a Microsoft Word certificate template
- Read candidate details from an Excel worksheet
- Automatically replace placeholders
  - Candidate Name
  - Certificate ID
  - Date
- Generate personalized certificates
- Export certificates as PDF
- Save all generated certificates to a selected folder
- Fast and fully automated process

---

## Tools Used

- Microsoft Excel

---

## Input Files

- Certificate Details.xlsx
- Certificate Template.docx

---

## Output

- Individual PDF certificate for each candidate
- Automatically generated without manual editing

---

## Workflow

1. Open the Excel workbook.
2. Select the Word certificate template.
3. Choose the output folder.
4. Click the **Generate Certificates** button.
5. The VBA script reads candidate details from Excel.
6. Placeholders in the Word template are replaced automatically.
7. Individual PDF certificates are generated and saved.
