# Certificate Generator using Excel VBA

## Overview

This project automates certificate generation using Excel VBA and Microsoft Word.

### Features

- Select Word certificate template
- Select output folder
- Automatically replace:
  - Candidate Name
  - Date
  - Certificate ID
- Generate separate PDF for each candidate
- Save PDFs using Candidate ID

## Technologies Used

- Excel VBA
- Microsoft Word Automation
- PDF Export

## Workflow

Excel Data → Word Template → PDF Certificate

## Output

One personalized PDF certificate is generated for each candidate.
