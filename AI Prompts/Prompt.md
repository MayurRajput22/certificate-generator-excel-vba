# AI Prompt Used

## About

This project was developed using AI-assisted prompt engineering. The generated VBA code was manually tested, debugged, and customized to correctly handle Microsoft Word text boxes, placeholder replacement, and PDF generation.

---

## Original Prompt

I have two files: A Word document containing a certificate template with three textbox: <<CANDIDATE NAME>>, <<DATE>>, and <<CERTIFICATE ID>>.

Second File: An Excel sheet with candidate details, structured as follows:

- Column A: Candidate ID
- Column B: Candidate Name
- Column C: Date
- Column D: Certificate ID

Write an Excel VBA code for me, where once I run the VBA code:

1. First, show a pop-up to select the certificate design Word file.
2. Next, show a pop-up to select the destination folder for saving PDFs.
3. The macro should replace the text boxes inside the Word document using the corresponding data from the Excel sheet.
4. Generate a separate PDF certificate for every candidate.
5. Save each PDF using the Candidate ID (e.g., 1001_Certificate.pdf).
